# Javaweb 图书管理系统 java760

## 项目概述

Javaweb图书管理系统是一个基于Java技术栈的Web应用程序，主要采用JSP、Servlet技术，结合MySQL数据库和Tomcat服务器，前端使用Bootstrap、HTML5及Layui框架进行界面构建。

## 技术栈

- 后端：Java、Servlet、MySQL、Tomcat、Maven
- 前端：Bootstrap、HTML5、Layui

## 功能模块

### 一、用户模块

- 查看公告
- 图书查看
- 个人违章信息（登录后）
- 图书馆读者留言（登录后）
- 查看借阅规则
- 个人信息（登录后）
- 个人借阅信息（登录后）

### 二、图书管理员模块

- 借还图书
- 借阅报表
- 还书报表
- 个人信息修改

### 三、系统管理员模块

- 书籍管理
- 分类管理
- 借阅证管理
- 借阅规则管理
- 借阅信息查询
- 图书管理员管理
- 图书借阅统计

## 系统角色

- 用户
- 图书管理员
- 系统管理员

## 运行环境

- 数据库：MySQL
- 服务器：Tomcat
- JDK版本：需符合项目配置要求

## 部署步骤

1. 环境配置：确保Java、MySQL、Tomcat等必备软件安装正确。
2. 数据库构建：根据项目需求，建立相应的数据库和表结构。
3. 项目编译：使用Maven进行项目编译，生成可部署的WAR包。
4. 部署应用：将WAR包部署到Tomcat服务器上。
